# Problem Artifact Templates

This directory holds first-pass Builder artifact templates for real-world problem support.

Current packs:

- `legal/contents-certification-template.md`
- `legal/complaint-filing-template.md`
- `legal/evidence-index-template.md`
- `financial/debt-summary-template.md`
- `financial/creditor-map-template.md`
- `financial/program-eligibility-checklist-template.md`

Usage notes:

- Replace bracketed placeholders like `[USER_NAME]` and `[DATE]`.
- Keep source facts, dates, amounts, and counterparties consistent across all artifacts.
- These are working templates for case preparation, not legal or financial advice by themselves.
