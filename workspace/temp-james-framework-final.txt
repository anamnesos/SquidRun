CONVERGED LEARNING FRAMEWORK (Oracle + Architect agreed)

Oracle said: "Build the minimum system that forces us to learn from whether our calls actually worked." 3 things, not 6.

1. PREDICTION TRACKER (highest impact — build first)
Every time we call BUY or SELL with conviction 0.55+:
- Log it: timestamp, coin, direction, price, reasoning
- Auto-check at 2h, 4h, 8h, 24h: did price move our way?
- Track accuracy by setup type (continuation longs, peak fades, flush shorts, etc.)
- This forces honesty and exposes bias over time

2. MACRO-FIRST HEADER (start every cycle with 4 lines)
Before looking at ANY individual coin:
- Where is BTC vs its key level?
- Is the tape risk-on or risk-off?
- Any active news/event?
- Are we trading WITH or AGAINST the tape?
This would have caught today's BTC $74k rejection = macro dump signal

3. POST-MORTEM TAGGING (every miss gets ONE root cause tag)
When a call was wrong, tag it with exactly one reason:
- "early short" / "late chase" / "ignored macro" / "wrong timeframe" / "crowding misread" / "bad management"
- Without this the prediction log is trivia, not learning
- Over time the tags show us our #1 failure pattern

WHAT WE REMOVED (to avoid another rule maze):
- No forced cross-agent challenge on a schedule (instead: challenge when confidence >0.6 AND call repeats 3+ cycles)
- No automatic alarm if accuracy <50% without segmenting by type
- No buyback cross-ref on every single coin (keep for watchlist only)

WHAT WE'RE ADDING (tools/data):
- BTC key-level monitor (first thing checked each cycle)
- Prediction log with auto-scorer
- Simple tape-state indicator (BTC + ETH + SOL + top-beta breadth)
- Unlock calendar cross-check from Tokenomist

PRIORITY ORDER:
1. Prediction tracker (highest impact)
2. Macro-first header (prevents today's biggest miss)
3. Miss-tagging (turns losses into learning)
4. Unlock monitor
5. James's pattern notes reference

Oracle's exact words: "Do not build a giant process machine. Build the minimum system that forces us to learn."

We'll start building the prediction tracker tonight if you want. It's the one thing that changes everything — actual scored accuracy by setup type, tracked over days.
