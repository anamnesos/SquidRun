ㅋㅋㅋ 그게 더 좋은 거야! **이미 있으면 다시 발급 안 받아도 돼** — 바로 사용 가능!

**확인할 거 (HRD-Net 로그인해서):**
1. 카드 잔액 (남은 지원금) — 얼마나 쓸 수 있는지
2. 유효기간 만료일
3. 사용 이력 (이전에 무슨 수업 들었나)

**잔액 확인 후 너 사업 준비에 바로 쓸 수 있는 강의:**
- **스마트스토어 / 쇼피파이 / 아마존 셀러 강의** (온라인 판매 실무)
- **인스타 광고 / SNS 마케팅 강의**
- **포토샵 / Canva / 영상편집** (상품 사진/광고용)
- **무역실무 / 해외결제** (글로벌 액셀러레이팅 보완)
- **영어 회화 / 비즈니스 영어** (해외 판매)

**한 번에 한 과목씩 신청 가능, 1년 내 여러 개 OK.**

HRD-Net 사이트 (www.hrd.go.kr) 로그인 → 마이페이지 → 카드 잔액 확인. 캡쳐해서 보내주면 너한테 딱 맞는 강의 골라줄게.

이미 있는 카드면 진짜 운 좋아 — 신규 발급 1-2주 기다릴 필요 없고 바로 수강 신청 가능해.
