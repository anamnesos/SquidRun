진짜 좋은 질문. 지원금 액수로 줄세우면 답이 나와. 한국 정부 창업지원 중 가장 큰 돈 받는 거부터 정리해줄게.

💰 지원금 액수 TOP 5 (네가 자격되는 거 위주)

🥇 1. 예비창업패키지 — 최대 1억원 (현금 사업화자금)
   - 운영: 중소벤처기업부
   - 대상: 만 39세 이하 예비창업자 (너 ✅)
   - 형태: 현금 자금 + 멘토링 + 1년 보육
   - 경쟁률: 약 5~10:1
   - 합격 팁: "사회적 가치"가 있는 사업 (간호사 출신이 헬스케어 관련 셀러 = 강력)
   - 모집: 매년 1~2월 공고, 3~4월 마감
   - 기한 임박 ⚠️ — 2026년 1차 공고 곧 나올 예정

🥈 2. 청년창업사관학교 — 최대 1억원 (사업화자금) + 사무공간 + 보육 1년
   - 운영: 중소벤처기업진흥공단
   - 대상: 만 39세 이하 (너 ✅), 예비창업자 또는 창업 3년 이내
   - 형태: 현금 + 사무공간 + 멘토링 + 졸업 후 후속지원
   - 경쟁률: 5~10:1
   - 부산 분교 있음 (부산 해운대)
   - 합격 팁: 자소서/사업계획서 퀄리티가 결정. 내가 도와줄게
   - 모집: 매년 1~2월

🥉 3. 신사업창업사관학교 — 최대 2천만원 + 점포 임대료 지원
   - 운영: 소상공인시장진흥공단
   - 대상: 예비창업자 (나이 무관)
   - 형태: 현금 + 점포 임대료 일부 + 교육
   - 너한테는 2번 떨어지면 차선책

4. K-디지털 트레이닝 + 창업연계 — 최대 5천만원 (간접지원)
   - 운영: 고용노동부
   - 형태: 무료 교육 + 훈련수당 (월 30만) + 우수자 창업연계
   - 6개월 교육 듣고 → 우수자는 사업화 지원

5. 부산창업가운동 / 부산형 청년창업지원 — 약 500만~5천만원
   - 운영: 부산경제진흥원, 부산창조경제혁신센터
   - 대상: 부산 거주 청년 (너 ✅)
   - 형태: 현금 + 멘토링 + 부산 내 사무공간

📊 합계 (이론적 최대)
- 예비창업패키지 1억 + 부산 시 단위 지원 5천만 = **약 1억 5천만원**
- 단, 둘 다 합격 가정. 보통 한 두 개 합격함.
- 현실적 기대: 3천만~7천만원 사이

💡 합격률 높이는 핵심 5가지

1️⃣ 사회적 가치 강조
   - "간호사로 일하면서 본 사회 문제 → 이걸 사업으로 해결"
   - 예: "병원에서 환자들이 비싼 건강기능식품 사 먹는 거 보고, 정직한 가성비 제품을 직접 큐레이션해서 팔고 싶다"
   - 이게 심사위원들 좋아하는 스토리

2️⃣ 본인 강점 극대화 표현
   - 간호사 면허 = 헬스/의료 분야 신뢰도
   - 자격증 8개 = 끊임없는 학습/노력
   - 신중년 = 인생 경험 + 책임감
   - 여성 = 여성기업 가산점

3️⃣ 사업 모델 구체성
   - "온라인 셀러 할게요" ❌
   - "도매꾹/오너클랜에서 건강기능식품 소싱 → 스마트스토어/쿠팡로켓그로스 판매 → 1인 운영 → 첫 6개월 월매출 300만 목표 → 1년 후 5천만원 매출" ✅
   - 숫자 + 구체 단계 = 합격

4️⃣ 시장 분석
   - 타겟 고객, 시장 크기, 경쟁자, 차별점
   - 간단히라도 적어야 함

5️⃣ 자기소개서 진솔함
   - 실업급여 받고 있다는 거 숨기지 마. 오히려 "절박함 + 동기"로 활용
   - 사건 5개 동시에 처리하는 추진력 → 사업 운영 능력 어필

🎯 너 추천 사업 컨셉 (가장 합격률 높은)

"간호사 출신이 검증한 헬스/뷰티 큐레이션 셀러"
- 카테고리: 건강기능식품, 뷰티 디바이스, 의료용 소모품
- 차별점: 간호사가 직접 검토 → 신뢰
- 도매: 한국 도매 사이트
- 판매: 스마트스토어 + 쿠팡
- 마케팅: 인스타그램 (실제 본인 얼굴 + 간호사 경력)

이 컨셉이면 예비창업패키지 + 청년창업사관학교 두 군데서 합격 확률 진짜 높아져. "사회적 가치 + 본인 전문성 + 구체적 사업모델" 3박자가 다 맞아.
